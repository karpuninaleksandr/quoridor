Коридор
=======

Как запустить
-------------

1. Распакуйте архив в любую папку.
2. Запустите приложение двойным кликом:
   - Windows: Quoridor.exe
   - macOS: Quoridor.app
3. После запуска откроется окно браузера со стартовой страницей игры.
4. Чтобы полностью закрыть приложение, закройте небольшое окно "Коридор".

Если браузер не открылся автоматически
--------------------------------------

Нажмите кнопку "Открыть игру" в небольшом окне "Коридор".

Важно
-----

Папку приложения не нужно переименовывать или разбирать на отдельные файлы.
Java устанавливать отдельно не требуется: она уже включена в сборку.
