# Сборка приложения для запуска двойным кликом

Приложение можно упаковать в desktop-версию через `jpackage`. В такой сборке Java Runtime лежит внутри папки приложения, поэтому научнику или рецензенту не нужно устанавливать Java и не нужно открывать командную строку.

## Windows

Сборку Windows-версии нужно выполнять на Windows с установленными JDK 17+ и Maven. Скрипт собирает приложение в production-режиме Vaadin, поэтому при первом запуске сборки Maven может скачать frontend-зависимости.

```powershell
.\scripts\package-windows.ps1
```

Результат:

```text
dist\windows\Quoridor-windows.zip
```

Для передачи другому человеку нужно отправить архив. После распаковки внутри будет папка:

```text
dist\windows\Quoridor
```

Пользователь распаковывает архив и запускает `Quoridor.exe` двойным кликом.

## macOS

Сборку macOS-версии нужно выполнять на macOS с установленными JDK 17+ и Maven. Скрипт собирает приложение в production-режиме Vaadin, поэтому при первом запуске сборки Maven может скачать frontend-зависимости.

```bash
./scripts/package-macos.sh
```

Результат:

```text
dist/macos/Quoridor-macos.zip
```

Для передачи другому человеку нужно отправить этот архив.

## Как работает desktop-режим

При запуске desktop-версии приложение:

1. запускает локальный Spring Boot/Vaadin сервер на свободном порту;
2. открывает стартовую страницу игры в браузере;
3. показывает маленькое окно управления;
4. завершает сервер при закрытии этого окна.

Если браузер не открылся автоматически, в окне управления есть кнопка `Открыть игру`.
